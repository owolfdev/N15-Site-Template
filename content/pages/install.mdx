Install [MDXBlog Basic](https://github.com/owolfdev/next-template-mdx-shad), a more spare and stable version of the codebase, which is a good starting point for a simple blog or to learn how to build a blog with MDX and Next.js. This version uses the @next/mdx package, the app router, and more.

Install [MDXBlog](https://github.com/owolfdev/mdx-blog), this site&apos;s codebase, including all the content, all the tools we use daily, as well as any issues that we are working to improve. This is the evolving template, and the latest version of the MDXBlog. This version currently uses the @next/mdx package, the app router, and more.
